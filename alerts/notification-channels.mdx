---
title: 'Notification Channels'
---

Nevua Markets supports a number of notification channels, so that new markets and price movements find you where you are

## In App 
- Receive notifications directly in the Nevua Markets application
- Always enabled by default
- Real-time notifications while you're using the app

## Telegram 
- Receive notifications via Telegram bot
- Requires connecting your Telegram account
- Great for mobile notifications

## Discord
- Receive notifications in your Discord server
- Requires setting up a Discord webhook
- Perfect for team monitoring

## Webhook
- Send notifications to your own application
- Requires configuring a webhook endpoint
- Best for automated systems and integrations
